A Matlab toolbox to analyze slip transfer through grain boundaries.
=====================================================================

To get started in Matlab, cd into the folder containing this README file.
Then add the package path to the Matlab search path by typing "path_management".
Finally you can start the launcher by typing "demo" at the Matlab command prompt.

See the documentation (./doc) for further details.

Written by D. Mercier and C. Zambaldi, based on previous work of T.R. Bieler and C. Zambaldi.

Parts of this work were supported under the Materials World Network program (DFG ZA 523/3-1 and NSF-DMR-1108211). We also acknowledge useful discussions with Y. Su, P. Eisenlohr and M. Crimp.
